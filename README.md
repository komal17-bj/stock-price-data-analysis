
📈 Stock Market Analysis & Interactive Dashboard

🔎 Overview

This project analyzes historical stock market data to identify trends, volatility patterns, and relationships between financial assets.
The objective was to transform raw time-series data into structured insights using statistical analysis and interactive visualization.



💼 Business Relevance

Identified trend direction using moving averages
Evaluated stock volatility through closing price analysis
Compared time-based performance via resampling (daily, weekly, monthly)
Measured inter-stock relationships using correlation analysis
Built an interactive dashboard for exploratory decision-making
This project demonstrates practical application of data cleaning, time-series analysis, and visualization for financial insights.



🛠 Tech Stack

Python | Pandas | NumPy | Matplotlib | Seaborn | Streamlit | Jupyter Notebook



📊 Key Analytical Components

Time-series trend analysis
Rolling moving averages (short vs long-term signals)
Resampling & aggregation for temporal comparison
Multivariate analysis
Correlation matrix & heatmap visualization



🖥 Interactive Dashboard (Streamlit)

Features:
Dynamic stock selection using radio buttons
Real-time plot rendering
Correlation visualization
User-friendly layout for exploratory analysis


Run locally:

streamlit run app.py
